在 Windows Azure 中创建网站 (OS X)

概述

在此动手实验中，您将学习如何使用预打包应用程序库中的模板在 Windows Azure 中创建新网站。网站创建后，您将使用文本编辑器来编辑网站并使用 FTP 来更新网站。

目标

在此动手实验中，您将学习如何执行以下操作：

  • 使用模板库中的预打包应用程序在 Windows Azure 中创建新网站
  • 使用文本编辑器编辑文件来更新网站，并使用 FTP 来更新网站

先决条件

下面是完成此动手实验需要满足的先决条件：

  • 启用了网站预览的 Windows Azure 订阅 - 注册以免费试用

注意：本实验设计为使用 Mac OS X 操作系统

练习

此动手实验包括以下练习：

  1. 练习 1：使用应用程序库创建 Windows Azure 网站
  2. 练习 2：使用文本编辑器在 OS X 上自定义网站，并使用 FTP 更新网站

练习 1：使用应用程序库创建 Windows Azure 网站

在本练习中，您将使用应用程序库中的预打包 Web 应用程序创建一个新网站。

  1. 转到 Windows Azure 管理门户，并使用与您的订阅相关联的 Microsoft 凭据登录。

    登录到 Windows Azure 管理门户

  2. 在命令栏上，单击“新建”。

    创建新网站

  3. 单击“网站”，然后单击“从库中”。

    从库创建新网站

  4. 在“博客”类别中选择 WordPress 应用程序。

    库中的 WordPress 应用程序

  5. 在“配置应用”页中键入一个可用 Url（例如 wpressSample），然后从“数据库”下拉列表中选择“创建新的 MySQL 数据库”。

    配置网站

  6. 在“创建新的数据库”页上，键入数据库名称，或者使用门户提供的默认名称。选中此复选框接受法律条款，然后单击确认按钮以完成创建网站。

    配置数据库设置

  7. 创建网站后，您就可以看到“网站”部分中列出了此网站。

    WordPress 网站

  8. 单击网站打开其仪表板，然后单击“网站 URL”链接以浏览至新网站。

    WordPress 网站仪表板

  9. 现在将配置 WordPress 在 Windows Azure 上运行。填写“网站标题”、“用户名”、“密码”和“电子邮件”字段。然后，单击“安装 WordPress”。

    安装 WordPress 应用程序

  10. 将开始安装 WordPress。一旦安装过程完成，就会打开一个屏幕，其中显示最终状态。单击“登录”以访问 Word Press 应用程序。

    成功安装

  11. 输入先前配置的“用户名”和“密码”，然后单击“登录”以打开管理网站。

    WordPress 登录

  12. 要打开 WordPress 博客，在管理网站上，您可以单击具有该网站名称的链接（位于页面的左上角），在此情况下是 WordPress 示例。

    WordPress 管理网站

  13. 此时将显示示例“Hello World”博客文章。在下一个练习中，您将学会如何使用 FTP 发布功能来自定义博客页。

    WordPress“Hello World”示例博客

练习 2：使用文本编辑器在 OS X 上自定义网站，并使用 FTP 更新网站

在此练习中，您将使用 FTP 发布功能来更新网站，在博客文章中添加一个 Facebook 的 Like 按钮。

  1. 转到 Windows Azure 门户仪表板并复制“FTP 主机名”。

    获取 FTP 部署主机名

  2. 回到命令提示符，并通过运行以下命令连接到 FTP 发布服务（将主机名替换为从门户获得的值）。

    CommandPrompt
    ftp waws-prod-blu-001.ftp.azurewebsites.windows.net

    连接到 FTP 服务器

    注意：将第一个 waws-prod-blu-001.ftp.azurewebsites.windows.net 值替换为从门户中获得的值。此外，还删除上述的 ftp:// 前缀。

  3. 提供部署凭据的“用户名”和“密码”。确保“用户名”前面有“网站”的名称（例如，wpressSample\admin）

    提供 FTP 凭据

    注意：部署凭据不同于与您的 Windows Azure 订阅关联的 Live ID，并适用于与您的订阅关联的所有 Windows Azure 网站。如果您不知道自己的部署凭据，使用管理门户可以轻松重置它们。打开网站的“仪表板”页，单击“重置部署凭据”链接。提供一个新密码，然后单击“确定”。

    输入用户名和密码

  4. 转至 wp-content | themes | twentyeleven 文件夹，使用 GET ftp 命令下载 single.php。

CommandPrompt
   get site/wwwroot/wp-content/themes/twentyeleven/single.php single.php

下载 single.php

  1. 在文本编辑器中打开 single.php。

    打开 single.php

  2. 将以下代码添加到 <div id="content" role="main"> 元素之前。保存该文件。

    JavaScript
    <div id="fb-root"></div> <script src="http://connect.facebook.net/en_US/all.js#xfbml=1"></script> <fb:like href="<?php echo get_permalink(); ?>" show_faces="true" width="450"></fb:like>

    注意：此代码将包含 JavaScript SDK (JS SDK)，使您可以使用 Facebook 社交插件。您可以在此处了解有关 Facebook 的“Like”按钮的更多信息。

    添加 Facebook 的 Like 按钮

  3. 使用 PUT ftp 命令上载 single.php 文件。

    CommandPrompt
    put single.php site/wwwroot/wp-content/themes/twentyeleven/single.php

    上载 single.php 文件

  4. 现在，打开 WordPress 博客，以查看您刚刚添加的 Facebook 的 Like 按钮。

    Facebook 的 Like 按钮

小结

在此动手实验中，您学习了如何使用预打包应用程序库中的模板在 Windows Azure 中创建新网站。网站创建后，您使用文本编辑器更新了网站，并使用 FTP 更新了网站。
